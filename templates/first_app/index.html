House Keeping ERP WebApp

(Showing use of Restful API developed using Python Django)
This is a Django based webapp. It is developed keeping in mind the housekeeping management service use case i.e. to manage details of the assets, workers and tasks of such an organisation.





URLs of importance: -

  • localhost/first_app/index - It is the home page of webapp

  • localhost/first_app/add-asset/ - API 1

  • localhost/first_app/add-task/ - API 2

  • localhost/first_app/add-worker/ - API 3

  • localhost/first_app/assets/all/ - API 4

  • localhost/first_app/allocate-task/ - API 5

  • localhost/first_app/get-tasks-for-worker/{workerid} - API 6

    • => example->http://127.0.0.1:8000/first_app/get-tasks-for-worker/w1
